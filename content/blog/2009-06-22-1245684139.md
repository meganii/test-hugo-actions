```
for((i=0;i<5;i++));do java Ga_binary > R"$i".dat;done 
```

シェルスクリプトの変数名には慣習的に大文字が使われることが多い。使い捨ての変数は小文字、それ以外は大文字を使用するのが、メジャーみたい。

これで、毎回処理が終わるまで待たなくても結果を得ることが出来る。
